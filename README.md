# Dotfiles

## Installation

**Single command line:**
```bash
cd ~ && git clone git@github.com:2rage/dotfiles && cd dotfiles && .\bootstrap.ps1
```
**Separate commands:**
```bash
cd ~
git clone git@github.com:2rage/dotfiles
cd dotfiles
.\bootstrap.ps1
```
